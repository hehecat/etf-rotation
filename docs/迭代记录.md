## 已验证有效

1. **趋势过滤(大盘MA20以下空仓)** — 最关键, V1→V2从-43%到+84%
2. **纯动量因子** — 比多因子更有效
3. **过热过滤(20日>30%不追)** — 避免接飞刀
4. **迟滞过滤(新候选高15-20%才换)** — 降低换手
5. **持1只优于持2-3只** — 跨行业轮动中成立
6. **双均线过滤(MA10+MA20)** — 进一步降回撤

## 已验证无效/弱

1. 风险调整动量(Sharpe) — 稀释进攻性
2. 加速度动量 — 追高
3. 最大持仓期限制 — 几乎无效果
4. 移动止损 — 帮助有限
5. 综合多因子(7个等权) — 不如简单动量
6. 同行业ETF内部轮动 — 无意义,高度相关

### V8 核心结论

1. **长窗口+低频=最佳**: lookback 60日 + 调仓15天 + 双均线 = 夏普4.5
2. **效率因子被低估了**: 纯效率(净移动/总波动)夏普4.1, 之前没用过
3. **持2只比1只好**: 35只ETF池中, 持2只大幅降回撤
4. **高迟滞有帮助**: 40-50%迟滞比10-15%更好——管住手
5. **移动止损几乎无用**: 跨行业ETF池中, 趋势过滤比移动止损更有效
6. **市场宽度有效**: 20-40%ETF上涨才入场, 可以提高稳定性

### V10 抗过拟合检验

| 检验 | 方法 | 关键发现 |
|------|------|----------|
| Walk-forward | 4折 train选参→test | 选参OOS均收益+3.6%, 正折3/4 |
| 固定配置OOS | 各配置在4个test窗 | C01: +3.4%/折, 3/4正; B10: +10.5%但前折差 |
| 成本敏感 | 万0.5 → 万5+滑点万10 | **C01最稳**: 极端仅-14.5pp; 周频B10 -26.7pp |
| 参数邻域 | rb/hyst/mh/权重扰动 | **C01: 21/21全正, CV=0.21**; C13: 17/17, CV=0.29 |

#### 如何解决过拟合 (已落地)

1. **样本外优先**: 不以全样本夏普定生死, 看 walk-forward / 时间切分 / 换池
2. **简单优先**: 拒绝「全能堆参数」; C01 仅 eff+mtf+15d
3. **成本加压**: 低频策略对滑点不敏感 → 更可信
4. **邻域平坦**: 参数小扰动不崩溃 → 非刀锋最优
5. **固定后冻结**: 生产配置至少跑3个月; 连续2个调仓周期失效→降仓, 不立刻重优化
6. **警惕短窗年化**: 单折OOS夏普可被极短区间放大, 看正折比例+最差折

#### 生产上线
- `etf_signal_v2.py` → **C01_效15d** (eff0.6+mtf0.4, rb=15, top_n=1, MA20趋势, 止损-8%, 迟滞20%, 最小持仓5日)
- 佣金对齐回测: 万0.5

### 更长样本如何获取 (C01 已冻结, 备查)

当前 `stock-api get-klines` **支持长历史**, 之前回测只用了 `count=500`(~2年)。

```bash
# 日线最长实测
npx -y stock-api get-klines SH510300 --period day --count 2500 --adjust qfq
# → 3436根, 2012-05-28 ~ 2026-07-17

# 行业ETF受上市日限制, 不是接口上限
# SH512760 芯片: 2019-06 起 (~1722根)
# SH515120 创新药: 2021-01 起 (~1341根)
# SH588000 科创50: 2020-11 起
```

| 要点 | 说明 |
|------|------|
| 接口 | `count` 提到 2000~3000 + `--adjust qfq`(前复权) |
| 真实瓶颈 | **ETF上市日**, 不是API; 池子交集常从最晚上市的那只开始 |
| 做长样本WF | 应用「共同交易日交集」或「允许个股缺失、用可得子集」 |
| 周线 | `period week --count 500` 可覆盖更久但信号频率变了 |
| 其它源 | ashare-mcp / 东财 / tushare(需token) 可补, 目前 stock-api 已够用 |

**C01 冻结纪律**: 生产信号不因长样本结果改参; 长样本仅用于事后验证报告。

## 详细迭代记录

### V1 基线
- 因子: mom20 25% + mom5 15% + vol 20% + vol_trend 20% + ma_dev 20%
- 结果: -43%, 回撤-61%
- 问题: 大盘下跌时满仓硬扛

### V2 趋势过滤
- 新增: 沪深300 < MA20 → 空仓; 止损冷却5天; 过热过滤
- 结果: +84%, 回撤-12%
- 结论: 趋势过滤是核心风控

### V3 因子权重
- 纯动量(50+30+20) 最优 +134%
- 低波主导 胜率最高但收益一般

### V4 参数网格(144组)
- 最优: 调仓7天/持1只/迟滞15%/止损-10% → +209% 夏普7.0
- 但池子含3只创新药高度相关 → 过拟合风险

### V5 过拟合检验
- 原池 +209%, 行业ETF池 -41%, 宽基 +43%
- 结论: 收益严重依赖池子构成

### V6 跨行业池
- 重建真正跨行业10赛道池
- 最优: 调仓5天/持1只/迟滞20% → +141% 夏普3.2
- 换池子仍正收益 → 通过基本稳健性检验

### V7 新因子
- 多周期一致性略胜纯动量
- 双均线过滤: 回撤-17.6%, 夏普2.7
- 风险调整动量/加速度动量表现差
